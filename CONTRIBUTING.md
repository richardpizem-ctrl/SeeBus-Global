# Pravidlá prispievania (Contributing Guidelines)

Ďakujeme, že chceš prispieť do projektu SeeBus‑Global!

## Ako nahlásiť chybu (Bug Report)
- Použi šablónu „Bug Report“ v Issues.
- Pridaj jasný popis problému, kroky na reprodukciu a screenshot/video ak je možné.

## Ako navrhnúť funkciu (Feature Request)
- Použi šablónu „Feature Request“.
- Popíš, prečo je funkcia užitočná a ako by mala fungovať.

## Ako poslať Pull Request
1. Vytvor si vlastnú vetvu (branch).
2. Urob zmeny a pridaj jasnú commit message.
3. Skontroluj, že projekt sa dá spustiť bez chýb.
4. Otvor Pull Request a vyplň šablónu.

## Štýl kódu
- Dodržuj existujúcu štruktúru projektu.
- Neposielaj veľké PR bez diskusie.
- Každá zmena musí byť prehľadná a odôvodnená.

## Komunikácia
- Buď slušný, vecný a rešpektujúci.
- Diskusie patria do GitHub Discussions.

Ďakujeme za každý príspevok!
